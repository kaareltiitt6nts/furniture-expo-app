# Expo React Native näidisrakendus

## Tutvustus

Lihtne mööblipoe React Native & Expo näidisrakendus.

## Struktuur

- `app/` -> rakenduse põhifailid
- `components/` -> komponendid
- `assets/` -> pildid ja muud taolised staatilised failid
- `global/` -> globaalsed muutujad

## Eeltingimused

- Node.js + npm

## Käivitamine

```bash
# 1. Klooni repo
git clone https://github.com/kaareltiitt6nts/furniture-expo-app.git

# 2. Mine kausta
cd furniture-expo-app

# 3. Paigalda sõltuvused
npm install

# 4. Käivita rakendus
npx expo start
```
